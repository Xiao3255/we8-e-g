# 前台

# 门户

左侧栏的门户是可以点击选择的，默认为公司门户，点击下方的其他门户选项，可以进行切换。

门户的顺序为后台管理员设置，本文书写时第一个门户选项为公司门户

# 流程中心

流程中心，下方共有待办事宜、已办事宜、办结事宜、我的请求、抄送事宜共五个选项，流程中心本质是一个筛选器，流程中心显示的所有信息，均可以在流程模块中被找到。

  • 待办事宜

指代未办的流程，含有用户创建后保存的暂未发起的流程，用户发起流程后被退回来的流程以及用户接收到的新流程。

  • 已办事宜

含用户已处理完的接收到的流程，以及用户自己发起后未归档的流程。

  • 办结事宜

用户发起后已经归档的流程，可以在此查阅。

  • 我的请求

用户发起后的流程可以在已办事宜中查到。

  • 抄送事宜

抄送给用户的流程。

上述均由后台管理员设置，如果实际操作中发现与文中描述存在差异，请联系后台管理员

# 未读文档

未读文档指的是最新的未阅读文档。

# 流程

点击流程，切换到流程新建页面。

# 新建流程

在此处可以进行新的流程发起，这里主要介绍下几个特殊的流程。

  • HR01-请假/调休申请流程

这个流程中有一个代理选项，代理指的是流程发起人是替代他人发起流程的，流程的请假人其实是代理单选框后填写的人。发起代理时一定要记得将代理的单选框勾上，这样代理功能才会生效。

  • FD09-工程项目类用款申请流程

这个流程中有一个模式选项，选择“常规”，则一切照常操作，选择“派驻人员”则需要用户手动填写“相应的负责人”以及“分管领导”，这两项均为可多选且必填项。

等等，关于其他流程的操作详情，可以直接通过发起流程时按实际情况操作。

# 流程操作

  • 提交

点击提交流程走向下一个节点，处在提交类型的节点，显示为提交，在节点上用户不可进行退回操作。

  • 提交至退回节点

点击后流程提交至退回前的节点。

  • 保存

点击后流程将会被保存，被保存的流程可以在流程中心的待办事宜处查看。

  • 批准

点击批准流程走向下一个节点，区别于提交类型的节点，在此节点上用户可进行退回操作。

  • 转发

将流程转发给其他人，转发不会影响流程走向。

  • 转办

将流程转办给其他人，转办不可二次转办。

  • 退回

将流程退回给流程中的其他人。

  • 加签

将流程进行加签，加签后需加签回复。

  • 删除

点击鼠标右键，选择“删除”进行流程删除操作。删除操作只有在用户自己的节点时才可操作。

注意

提交至退回节点、转发、转办、退回、加签等权限均由后台管理员设置，如若发现流程没有相应的选项，请联系后台管理员。

# 人事

点击人事，切换到查询人员界面。

# 查询人员

输入相应的人员名称，可以得到对应的人员的基本信息，如分部，部门，联系方式等。如果需要按条件查询可以在进行相应的条件填写进行查询。

提示

使用人事的查询人员功能可以帮助你在没有通讯录的情况下查到系统内人员相应的联系方式。

# 我的卡片

显示用户本人的相关信息。

# 签到签退

用户在移动端（微信、企业微信等）签到，签到详细信息将会在此显示。默认情况下，用户只能看到自己的签到签退信息。如果需要查看其他用户的签到签退情况则需要在后台进行相应的权限设置。

若需要相应权限请联系后台管理员

# 知识

点击知识，切换到文档新建页面。

# 新建文档

首先需要选择相应的文档存储路径，然后按"+"图标进行文档新建。

# 我的文档

这里存储的为用户新建的文档含发布的（状态显示为正常），以及未发布的（状态显示为草稿）。

注意

有些文档目录存在可删除的默认共享，在这些文档目录下创建文档时，请确认默认共享的范围是否是符合所创建文档所需的共享范围，如果不符合还请手动删除。

# 会议

点击常用旁边的按钮选择会议

# 会议日历

用户在此处可以进行相应的会议室使用登记，登记后的信息可以在会议室使用情况查看。

# 会议室使用情况

用户可以在此处查看当前会议室的使用情况

具体的会议室信息由管理员在后台进行相应的设置

# 修改密码

修改密码有两种途径

  • 点击底部锁头样式的按钮进入密码更改界面按相应提示进行密码更改。
  • 点击人事，密码设置，进入密码更改界面按相应提示进行密码更改。

# 忘记密码或账号被锁定

用户遇到此种情况请联系后台管理员。

当前的OA密码要求密码需由字母、数字及特殊符号组成，当前OA的密码并没有做重复验证，所以理论上可以将新密码与旧密码设置为同一个密码。

# 个性化设置

点击底部的齿轮按钮进入个性化设置，此处仅介绍几个常用设置。

# 常用批示语

点击下方的“常用批示语设置”最右边的添加和删除按钮来进行常用批示语设置。常用批示语设置完后，可以在流程的“签字意见”的“批示语”中被找到。

# 私人组

私人组，是用户自定义的组，每个用户自己设定，只能自己使用。点击“人力资源”进入相应设置。

  • 新建

选择新建，输入相应的私人组名称，然后选择相应的人员在点击保存就可以了。

  • 导入

按照相应的提示，下载模板，并按提示填写模板，最后再点击“导入”即可。

  • 使用

在一些要求用户输入相应人员的表单(如上文提及的HR01)以及用户在进行了文档发起时，均可以在选择时点击“按定义的组”然后选择“私人组”，找到适合的组来进行操作。

私人组就是将一些常用的人力资源分在一个组中以方便用户的操作，相对于私人组的还有公共组，公共组由管理员所设。

# 帮助Q&A

# Q：如果我登陆页面后没有找到相应的选项该怎么办？

A：如果登录页面后，页面没显示流程，门户，知识等选项，请点击常用旁边的按钮手动选择。

# Q：如果我不小心把页面的某一个小插件删除了，我要怎么找回来？

A：请尝试点击鼠标右键的同步首页选项。

# Q：为什么我登录后门户旁边只有一项没有可供切换的其他选项？

A：这通常是管理员设置时账号的安全级别未设置造成的，请联系管理员。

# Q：我如何切换到我的子账号？

A：当用户有子账号时，仍是通过其主账号进行登录，然后点击用户名旁边的按钮进行相应的切换。

# Q：我的信息显示不正确？

A：用户发现自己的信息显示不正确时请联系管理员。

# Q：我的流程发起后发现出错我该怎么退回？

A：流程发起后，在审批人尚未查看时，可以点击鼠标右键选择强制收回，如果审批人已经查看，则无法再自行退回。此时请联系管理员。

# Q：我该如何删除流程？

A：当流程在自己的节点时，可以点击右上角的删除或者鼠标右键点击删除来进行删除操作。

# Q：我把流程加签了，我现在不知道流程在哪？

A：流程加签不同于流程转发，流程加签后流程将在被加签人那，请联系被加签人点击加签回复以使得节点重新回到发起加签人那里。

# Q：为什么我登录后系统提示我修改密码？

A：为了保证系统安全，OA系统后台的安全设置为指定时间内需要重新修改密码，按提示操作即可。

# Q：我突然上传不了附件？

A：请先确认自己当前所使用的浏览器为谷歌浏览器，（使用QQ浏览器、360浏览器等双核浏览器的用户，可以将浏览器切换为极速模式），如果依旧无法正常上传请联系管理员进行处理。
