# 权限管理

# 权限调整

当有人员离职，人员岗位变动等而需要对相应的账号进行变动时，权限调整便派上用场。

使用时左边输入需要调整的人员的名称，右边输入相应的转移对象，然后选择需要调整的权限，最后点击开始执行即可。

# 角色设置

角色设置可以通过设置一个语义化的角色名称，并通过设置相应的权限来使得普通用户能够拥有部分属于管理员才能拥有的权限。

# 权限查询

输入相应的用户名可以查询其所拥有的权限，点击明细可以进行具体查看。

# 矩阵管理

# 矩阵设置

设置相应的矩阵来满足现实需求，如不同的部门对应不同的管理者，当同一个流程需要满足不同部门的时，矩阵绝对是一个好帮手， 通过设置相应的矩阵，便无需为满足多个不同的需求，而进行一大堆的条件设置。

# 维护者

点击矩阵的人形按钮，你可为你创建的矩阵设置相应的维护者。

TIP

利用好矩阵可以大大提升你的工作效率，例如：当现实情况中发生管理者调动时，你只需更改相应的矩阵即可，无需再一个个的进行相应的修改。

# 路径模板

此处存储的是流程模板，便于流程复用。

Heads Up!

使用流程模板虽然便于复用，但其所引用的表单与原流程是同一个表单，更改该表单字段将会对所有引用该表单的流程产生影响，建议仅将流程模板应用于相似度较高的流程，并在需要新的字段时直接进行新建操作，而不通过修改已有字段的名称。如所需字段为"项目名称"，已有字段为 “项目”，此时应该首选新建字段而不是修改已有字段，除非你自己清楚这样操作的后果。

# 表单管理

表单管理共有基本信息，编辑字段，字段显示名，行字段规则，列字段规则共5个选项。

  • 基本信息：可以进行流程的基本信息的查看包括表单名称、数据库表名、所属机构以及表单描述。
  • 编辑字段：新建的表单为空表单，可以在这里进行字段的添加。
  • 字段显示名：当对多语言有要求时，可以在这里设置其相应的显示名。
  • 行字段规则：利用系统内置的规则公式来快捷的设置处在同一行的字段。
  • 列字段规则：利用系统内置的规则公式来快捷的设置处在同一列的字段

# 流转设置

流转设置共有图形编辑，节点信息，出口信息共3个选项。

  • 图形编辑：点击编辑进入编辑器，编辑流程的图形界面（需要先安装flash）。

  • 节点信息：将流程的信息以节点的方式进行展示，鼠标右键点击编辑进入编辑界面，可以对节点的名称、类型、属性、顺序进行相应的修改。

  • 出口信息：用于调整流程顺序以及条件设置。

# 高级设置

高级设置共有功能管理，流程计划，流程编号，督办设置，流程创建文档，子流程，显示属性联动，字段联动，流程存为文档，流程转日程，浏览数据定义，自定义报表共12个模块，详细信息请参考流程创建
